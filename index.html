Hagrid.css

Hagrid is a simple and clean CSS grid based on Simple Grid with new features and an open-source license, see the GitHub page of the project for more informations.

This page show you the grid and the "view per device", the two main components of the CSS available here.

-------

The grid:

On a screen with width >= 768px: The columns in a grid are side by side.

On a screen with width < 768px: The columns are one above the other.

The code for a grid is:

<div class="grid">
   <div class="col-1-3">
      /* Column's content */
   </div>
   <div class="col-2-3">
      /* Column's content */
   </div>
</div>

A "grid" class is where you can place your columns, a grid made 1 when it's done. Here you have a column with class="col-1-3" and another with class="col-2-3", 1/3+2/3 = 1, perfect!

You can have columns with different sizes on the same grid, Hagrid supports many sizes : 1, 2, 3, 4, 5, 6, 8, 10 and 12 (see all the columns on the top of the page).

When you want a column with width : 1/3, the width is nearly perfect on old browsers (like IE) and perfect on modern browsers width the CSS component calc().

Last but not least, you can do a grid with columns on the right. Just write your column with "right" at the end of the class name, like:

<div class="grid">
   <div class="col-1-2 right">
      /* Column's content */
</div> </div>
